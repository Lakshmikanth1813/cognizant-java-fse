# Git HOL 2 - Git Ignore

This project demonstrates how to use `.gitignore` to ignore unwanted files and folders such as `.log` files and `log/` directories.

## Objectives
- Explain git ignore
- Explain how to ignore unwanted files using git ignore

## What was done
- Created a `.gitignore` file to ignore:
  - All `*.log` files
  - The `log/` directory
- Created sample `.log` files and a `log/` folder to verify they are ignored by Git
- Verified using `git status` that ignored files do not appear as untracked
